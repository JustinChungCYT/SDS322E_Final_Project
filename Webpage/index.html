Welcome to the Predicting Property Value Project

Overview

The goal of this project is to analyze and predict the prices of houses in Melbourne based on multivariate factors.
Based on our explorotary analysis of the many variables surrounding housing prices, we will test a model built upon those variables to analyze whether or not they can predict housing prices.
